# Customer Sales Analysis Dashboard

## 📌 Project Overview
This project analyzes customer sales data and visualizes insights using **Seaborn** and **Plotly**.  
It includes both static plots and an interactive dashboard to explore sales trends, product performance, and regional distribution.
